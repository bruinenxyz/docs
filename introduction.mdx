## What is Bruinen?
Bruinen enables your platform to validate and connect your users' profiles from across the internet.
We offer simple integration with a variety of data sources, including Facebook, Amazon, and many more.

## How does it work?
Bruinen offers an easy-to-use, consistent interface to access your users' profiles.
You can connect, authenticate, and pull data from those accounts all from Bruinen's platform.
This is accomplished through a few key features:

<CardGroup cols={3}>
  <Card title="Bruinen API" icon="code" href="/api-reference">
    A RESTful API to create users, add sources, authenticate accounts, and request data
  </Card>
  <Card title="Bruinen SDK" icon="cube" href="/api-reference/bruinen-connect">
    A Javascript SDK to enable insanely fast integration with the Bruinen system
  </Card>
  <Card title="Bruinen Connect" icon="link" href="/api-reference/bruinen-sdk">
    A managed UI to allow users to authenticate their accounts directly in your app
  </Card>
